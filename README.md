# Telegram-бот для Railway

## Деплой на Railway

1. Создай новый репозиторий на GitHub (например `multiunit-bot`)
2. Загрузи все файлы из этой папки
3. Зайди на railway.app → New Project → Deploy from GitHub
4. Выбери репозиторий `multiunit-bot`
5. После деплоя добавь переменные окружения (Variables):

| Переменная | Значение |
|-----------|---------|
| TG_BOT_TOKEN | твой токен от BotFather |
| API_URL | https://multi-production-0cab.up.railway.app |
| BOT_SECRET | multiunit-bot-2026 |

## После деплоя

Бот будет работать 24/7 без компьютера!
